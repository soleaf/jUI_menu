jUI_menu Example

jUI_menu is a easy beautiful Javascript tool for combobox and menu have input value.
Therefore You can use this component to form on your site. Or Use navigating with submenu!

Use simple!

Only code need anchor(location for combobox or menu) and menu contents list.
Menu content list link to root menu(data-root on menu->data-id on root). submenu link by menu's data-sub.
